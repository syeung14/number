Program 5-2

Regula Falsi Algorithm

This applet demonstrates finding a the root of a function f(x) by the Regula Falsi Algorithm.

To run the demo:

  1. Choose a function.
  2. Press the Step button to single-step the algorithm.
  3. Press the Run button to animate the algorithm.
